Git Commit Tracking

Detects new commits every 30 seconds. Each commit appears in your project's activity log with hash and message.

File Change Detection

Watches for new and modified files in any folder — code, design assets, documents. Works with or without git.

Auto Progress & Status

Progress updates from task completion. Status transitions automatically — Planning to In Progress to Done.

Menu Bar App

Lives in your menu bar. Click the icon for a quick glance at all projects, or open the full dashboard.

Task Management

Add tasks, check them off. Progress and status update themselves. Markdown notes with live preview.

Works Offline

Everything runs locally on your Mac. No accounts, no cloud, no telemetry. Your data stays on your machine.

How it works

1

Create a project

Give it a name, set priority and due date. Add tasks if you want granular tracking.

2

Link a folder

Point it at any directory on your Mac — a git repo, a design folder, a docs directory.

3

Work normally

The app scans every 30 seconds. Commits, file changes, and tasks automatically update your project's progress.
