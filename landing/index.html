Features

Everything happens automatically

Link a folder. The app does the rest.

Git Commit Tracking

Detects new commits every 30 seconds. Each commit appears in your activity log with hash and message.

File Change Detection

Watches for new and modified files in any folder. Code, design assets, documents. Git optional.

Auto Progress & Status

Progress updates from tasks. Status transitions automatically from Planning to In Progress to Done.

Menu Bar App

Lives in your menu bar. Quick glance at all projects from a popover, or open the full dashboard.

Task Management

Add tasks, check them off. Markdown notes with live preview. Everything syncs to progress automatically.

Teams & Organizations

Create organizations, invite team members, assign roles. Admin, Member, and Viewer access levels built in.

Audit Log

Full activity trail showing who did what, when. Filter by user or project. Admin-only access for security.

Project Assignment

Assign projects to team members. "My Projects" view shows your work. See who's working on what at a glance.

Getting started

Three steps. Zero config.

Up and running in under a minute.

1

Create a project

Give it a name, set priority and due date. Add tasks if you want granular tracking.

2

Link a folder

Point it at any directory on your Mac. A git repo, a design folder, anything.

3

Work normally

The app scans every 30s. Commits, file changes, and tasks update progress automatically.
